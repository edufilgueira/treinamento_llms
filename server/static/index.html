      Configurações do modelo

      Ajusta o teu system prompt; ele junta-se ao prompt global (definido pelo administrador) em cada geração.

      Parâmetros de geração e llama-server

      Estes valores aplicam-se a todos os pedidos para o llama-server. O n_ctx aqui define o orçamento com que o Oráculo trunca o histórico antes de enviar (heurística ~4 chars/token). O binário deve arrancar com -c ≥ a este n_ctx (senão o llama-server pode devolver 400). Após alterar IP, porta ou «Usar llama-server», reinicia o processo Oráculo.
